# Fixed Income Modeling and Interest Rate Analytics

This repository contains a collection of Fixed Income and Interest Rate Modeling projects developed as part of coursework and quantitative finance practice. The notebooks implement core techniques used in fixed income analytics including yield curve construction, credit spread estimation, term structure modeling, interest rate simulation, and quantitative interest rate models.

The projects demonstrate practical implementations of yield curve estimation methods, Nelson–Siegel–Svensson term structure modeling, numerical optimization techniques, spline interpolation, and stochastic interest rate modeling using QuantLib.

# Quantitative Finance Concepts Demonstrated

yield curve construction, term structure modeling, credit spread analysis, Z-spread estimation, Nelson–Siegel–Svensson modeling, spline interpolation, nonlinear optimization, interest rate modeling, stochastic interest rate simulation, Hull–White model.


# Projects Overview

## Yield Curve Construction and Credit Spread Analysis

Constructed Treasury yield curve tenors and computed bond spreads including T-spreads and Z-spreads relative to benchmark Treasury rates and spot curves.

Methods / Models:
Nelson–Siegel–Svensson yield curve model, cubic spline interpolation, credit spread analysis, Z-spread estimation, yield curve construction.

Libraries:
pandas, numpy, scipy, matplotlib, datetime.

## Nelson–Siegel–Svensson Yield Curve Calibration

Calibrated the Nelson–Siegel–Svensson (NSS) term structure model to market yield data using nonlinear optimization techniques.

Methods / Models:
Nelson–Siegel–Svensson yield curve model, nonlinear optimization, curve fitting, cubic spline interpolation.

Libraries:
pandas, numpy, scipy, matplotlib.

## Term Structure Modeling and Yield Curve Estimation

Implemented a yield curve estimation pipeline using NSS parameters and optimization techniques to generate spot rates across maturities.

Methods / Models:
Nelson–Siegel–Svensson model, yield curve fitting, numerical optimization, term structure estimation.

Libraries:
pandas, numpy, scipy, matplotlib.

## Interest Rate Modeling and Yield Curve Regression

Extended term structure analysis using regression-based techniques and financial analytics to analyze interest rate dynamics and curve behavior.

Methods / Models:
Nelson–Siegel–Svensson model, linear regression, numerical root-finding, yield curve analysis.

Libraries:
pandas, numpy, scipy, scikit-learn, matplotlib.

## Hull–White Interest Rate Simulation

Constructed interest rate simulations using the Hull–White short-rate model and generated stochastic interest rate paths.

Methods / Models:
Hull–White interest rate model, stochastic interest rate simulation, term structure modeling, Nelson–Siegel–Svensson yield curve generation.

Libraries:
QuantLib, pandas, numpy, matplotlib, scipy.
